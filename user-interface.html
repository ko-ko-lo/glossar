User Interface (UI)

2.7.2023

Die Aufgabe von User Interface Expert:innen ist es, Benutzern eine intuitive und benutzerfreundliche Erfahrung zu bieten und bezieht sich auf den visuellen und interaktiven Teil einer Software-Anwendung.

Aber was ist ein "User Interface"? (Nicht zu verwechseln mit einem User Experience).

Stellen wir uns vor, wir haben ein technisches Gerät, zum Beispiel ein Smartphone. Und dann haben wir eine Person, die dieses Smartphone per Fingerabdruck entsperrt. Möglich wird dies durch eine Schnittstelle, über die die Person mit dem Gerät interagiert.

Wir bezeichnen die Person als User und die Schnittstelle als Interface und sprechen daher von einem User Interface.

Das Look & Feel eines User Interfaces

UI bezieht sich unter anderem auf den visuellen Bereich des Produktes, mit dem User interagieren - neben dem Look & Feel ist auch die Interaktivität eines digitalen Erlebnisses wichtig.

UI Expert:innen gestalten das Aussehen der Benutzeroberfläche. Zum Beispiel Typografie, Farbe, Abstände, Grids, Icons oder auch Schaltflächen.

Fachbegriff Erklärung
Typografie Typografie bezieht sich auf die Auswahl und Anordnung von Schriftarten, Schriftgrößen, Zeilenabständen und anderen typografischen Elementen, um gut lesbaren und ästhetisch ansprechenden Text zu erstellen.
Farbe Farben werden verwendet, um visuelle Hierarchie, Bedeutung, Stimmung und Ästhetik zu vermitteln. Durch die gezielte Verwendung von Farbe kann ein ansprechendes und konsistentes visuelles Design geschafft und gleichzeitig die Benutzerführung, Lesbarkeit und Markenerkennung verbessert werden.
Abstände Abstände helfen dabei, visuelle Klarheit und Organisation zu schaffen, indem sie Inhalte voneinander trennen und so die Hierarchie der Informationen verdeutlichen.
Grids Ein Grid ist ein Layout basierend auf Spalten und Zeilen, das verwendet wird, um UI Elemente auszurichten. Grids helfen ein konsistentes und ausgewogenes Layout zu erstellen.
Icons Icons sind kleine grafische Symbole oder Piktogramme ↪, die zur visuellen Darstellung von Aktionen, Funktionen oder Inhalten verwendet werden.
Schaltflächen Schaltflächen sind interaktive grafische Elemente, die es Benutzern ermöglichen, Aktionen auszuführen oder bestimmte Funktionen innerhalb einer Software-Anwendung auszulösen.

User Interface: Aufgaben und Aktivitäten

Neben empathischen Fähigkeiten, sich in die Lage des Nutzers zu versetzen, werden im User Interface Bereich besonders gute systematische Fähigkeiten benötigt. Den zu den Aufgaben gehört die Erstellung und Pflege eines Design Systems.

Ein Design System ist eine Bibliothek, eine Sammlung von Funktionen und visuellen Elementen. Es gibt eine klare Beschreibung, wie sich diese Elemente verhalten sollen, so dass ein einheitliches Design für alle Produkte gewährleistet ist. Das Designsystem fungiert als „Single Source of Truth”.

Auf Basis eines Design Systems werden User Interfaces und Prototypen gebaut. Prototypen werden für Qualitative Research, Quantitative Research und Usability Testing benötigt.

Zusammenfassung Aufgaben und Aktivitäten

  • Erstellung und Pflege eines Design Systems
  • Erstellung von User Interfaces und Prototypen
  • Erstellung von Design Dokumentationen

Welches Wissen wird benötigt?

Im User Interface wird ein umfangreiches Wissen benötigt. Neben technischen und gestalterischen Skills sind auch sehr gute Kommunikationsfähigkeiten gefordert. Oft sind User Interface Expert:innen die Schnittstelle zwischen Entwicklung und Produktmanagement und müssen die Anforderungen beider Parteien mit in die Arbeit einbeziehen.

Hier ist eine Auswahl der benötigten Skills:

  • Kenntnisse der Industriestandards
  • Verständnis für menschliche Faktoren
  • Kenntnisse in Layout, Form und Komposition
  • Grundlegende Kenntnisse in der Frontend-Entwicklung
  • Experten Kenntnisse in der Umsetzung von Prototypen

Fazit User Interface

Das User Interface (UI) bezieht sich auf die Benutzer-Schnittstelle einer Software-Anwendung, mit der Benutzer interagieren. Es umfasst visuelle Elemente wie Layouts, Schaltflächen, Icons und Farben sowie die Art und Weise, wie mit diesen Elementen interagiert wird.

Ein gut gestaltetes User Interface zielt darauf ab, eine intuitive, benutzerfreundliche und ansprechende Experience zu bieten.

„Design is not just what it looks like and feels like. Design is how it works.”
Steve Jobs
